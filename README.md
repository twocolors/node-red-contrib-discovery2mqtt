# node-red-contrib-discovery2mqtt
Nodes for controlling MQTT Discovery (Home Assistant) in Node-RED

More informations about [MQTT Discovery](https://www.home-assistant.io/docs/mqtt/discovery/)

Available nodes are:
* discovery2mqtt-in
* discovery2mqtt-out

Support component:
* switch
* sensor
